Introducción a JS

¿Qué es JS?

  • JS es un lenguaje de programación.
  • Creado por Brendan Eich en tiempo record de una semana.
  • Es el único lenguaje de programación que puede ser interpretado por el navegador.
  • El nombre tuvo algunas variantes, sin embargo JS se llama así ya que en ese tiempo estaba en auge Java.
  • JS fue mal visto desde su creación hasta el año 2009 aproximadamente.
  • Hasta 2009 se podía usar únicamente del lado del cliente. Actualmente con Node.js se puede ejecutar desde un servidor

Historia de JS

  • 1995 - Netscape crea JS
  • 1997 - Netscape se lleva JS a ECMA (European Computer Manufacturers Association)
  • 1997 - Se lanza ECMA-262 y se crea ECMAScript 1.0
  • 1998 - ECMAScript 2: Ajuste con el estándar internacional
  • 1999 - ECMAScript 3
  • 2009 - ECMAScript 5
  • 2011 - ECMAScript 5.1
  • 2015 - ECMAScript 6

Tecnologías descendientes de JS

  • AJAX: Creada por Microsoft y tiene como objetivo obtener información del servidor sin necesidad de recargar la página.
  • jQuery: Es una librería ya que ayudaba con problemas de compatibilidad en navegadores.
  • nodeJS: Permite ejecutar JS en un servidor.
  • JSON: Se ocupan en casi todos los sitios web sustituyendo al documento XML.
